# Customer-service CICD

Ce projet met en oeuvre l'implémentation d'un microservice multi connecteurs dans le cadre de systèmes distribués.

## Fonctionnalités

- Implementation de l'API REST avec la documentation Swagger
- Implementation de l'API GrapgQL
- Implementation du web service SOAP
- Implementation du service gRPC
- Mise en place d'un pipeline CICD basée sur github actions
